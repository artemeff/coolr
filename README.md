### coolr

---

Coolr can generate beaty colors from strings. Inspired by [String to Color](https://github.com/brandoncorbin/string_to_color) and [Colos gem](https://github.com/artemeff/colos).

---

```erlang
> coolr:do("test").
% => "194F5D"
> coolr:do("coolr").
% => "1EC053"
> coolr:do("artemeff").
% => "CFA5FE"
> coolr:do("erlang", -25).
% => "99B967"
```
